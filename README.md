# DiceApp AAA

# 開発環境
* Android Studio 4.0
* Java 14.0.1

# 主な機能
* 「サイコロを振る！」ボタンを押すと2つのサイコロがそれぞれ
   1~6までの数字をランダムで表示する。
*  サイコロを振った時、サイコロの転がる音が出る。
*  サイコロを振った時、サイコロが左右に揺れるアニメーションが起こる。
